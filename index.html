Kas tai?

Kas yra Code Dojo? Coder Dojo - bendruomeniniu savanorystės pagrindu sukurti programavimo pradmenų mokymo klubai 7 – 17 m. vaikams ir jaunimui.

Coder Dojo (www.coderdojo.com) skatina vaikus/jaunimą ir jų tėvelius mokytis programuoti, kurti tinklapius, žaidimus ir kitais įdomiais metodais sužinoti daugiau apie technologijų pasaulį ir gal būt su tuo susieti savo profesinę ateitį. Juk tai - labai paklausi profesija, kurios specialistų Europoje ir visame pasaulyje trūksta.

Šiuo metu 57 šalyse veikia apie 700 Coder Dojo klubų, kurie yra surengę tūkstančius renginių.

Vaikai, atėję į Coder Dojo užsiėmimą, bus mokomi, kaip sukurti pirmąją interneto svetainę. Jiems padėti pasirengę Lietuvos IT įmonėse dirbantys programavimo profesionalai - Coder Dojo mentoriai.

„Tai gali būti pradžia, už kurios dalis vaikų susidomės, supras, kad prie kompiuterio galima ne tik žaisti, bet ir kurti. Galbūt vieną dieną vienas iš šių vaikų sukurs Google dydžio įmonę. Lietuvoje. Tai yra didžiausia mano svajonė”, - sako Coder Dojo tarptautinės iniciatyvos ambasadorius Lietuvoje ir šios idėjos įgyvendinimo autorius EP narys Antanas Guoga.

Coder Dojo Lietuva iniciatorius, praktiškai išmanantis skaitmeninų verslų principus, Europos Parlamento narys Antanas Guoga:

Šiandien rytojaus IT sektoriaus darbuotojai dar sėdi mokykliniuose suoluose. Galime kartu imtis veiksmų, kad jie tobulėtų jau dabar. Vaikai imlūs moderniosioms technologijoms. Gali būti, kad pasimokę programuoti, jau už kelių metų jie imsis savo skaitmeninio verslo. O galbūt vieną dieną vienas iš šių vaikų sukurs Google dydžio įmonę. Lietuvoje. Tai yra mano didžiausia svajonė.

Dabar laikai, kai profesijų neskirstome pagal lytis. Dabar laikai, kai moterys pačios save išlaiko. Mergaitės, mokykitės programuoti, nes programuotojai labai daug uždirba!

Mentoriai

Kas yra Coder Dojo mentorius?

CoderDojo mentoriai yra IT įgūdžių turintys žmonės, kurie padeda užsiėmimuose dalyvaujantiems vaikams ir jaunimui mokytis. Kadangi visi Coder Dojo užsiėmimai yra paremti savanorystės principu, visi mentoriai yra pasikaustę skirtingose srityse ir būtent tų subtilybių moko vaikus ir jaunimą.

Mentoriai užsiima įvairia veikla – važiuoja į Coder Dojo užsiemimus skirtinguose Lietuvos miestuose, rengia medotinę medžiagą, rūpinasi interneto svetainės priežiūra, padeda organizuojant ir kuruojant užsiėmimus. Kiekvienas mentorius atranda tokią veiklą, kuri geriausiai atitinka jo įgūdžius ir suteikia jam daugiausiai smagumo

Pasidalink savo SUPERGALIOMIS ir prisidėk kuriant klestinčią Lietuvą! Užpildyk šią formą ( https://goo.gl/VSA0Aq) ir mes būtinai susisieksime.

Mūsų mentoriai

Šiuo metu Coder Dojo komandoje savanoriauja virš 40 mentorių, kurie yra tikri savo srities profesionalai ir siekia, jog programavimo SUPERGALIOMIS užsikrėstų kuo daugiau Lietuvos vaikų.

Susipažinkite su mūsų aktyviausiais mentoriais! Jei kyla klausimų – drąsiai į juos kreipkitės.

package lt.coderdojo.intro;

import java.lang.*;
import descr.Education;
import descr.programming.Lang;
import descr.work.Client;

public class CoderDojo {
  public CoderDojo() {};

  Education mokslas = new Education();
  mokslas.set("J. Snadeckio gimnazija");
  mokslas.set("Vilniaus Kolegija - Sisteminis programavimas");

  Lang[] programavimoKalbos =
    new Lang["Pascal", "Object Pascal", "Delphi", "C++", "C#", "JAVA",
             "JavaScript", "Python", "PHP", "Common Lisp (The Basics)"];

  Client klientas = new Client();
  klientas.set("BilietaiLT");
  klientas.set("Bankai"); // secure/slapta informacija
  klientas.set("Logistika");

  public void getPristatymas() {
    String rezultatas = "Trumpai apie save:";
    rezultatas += "\n" + mokslas.getList().AsString();

    for (int i = 0; i < programavimoKalbos.length; i++) {
      if (i == 0)
        rezultatas += "\n";
      else
        rezultatas += programavimoKalbos[i];
    }

    rezultatas += "\n" + klientas.getList().AsString;

    System.out.println(rezultatas);
    System.out.println("\n\n Placiau ziur. Linkedin: https://lt.linkedin.com/in/pavel-vrublevskij-04293763");
  }
}
